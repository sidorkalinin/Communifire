Communifire.iOS.App.Certificate.p12 is the certificate used by Communifire application send Push Notifications to respective clients.

Certificate was generated with help of .cer downloaded from https://developer.apple.com/account/ios/certificate/ for "com.axero.Communifire” as “Apple Push Services” type.

This .p12 file should be uploaded to /Assets/Uploaded-CMS-Files/Certificates folder so that Communifire application can make use of this certificate to send Push notifications.

For more information refer https://my.axerosolutions.com/spaces/33/axero-communifire-development/wiki/view/25163/getting-apns-certificate-communifire-ios-app-certificate-p12-for-push-notifications
